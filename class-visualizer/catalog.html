MIT Major Visualizer

What is this?

Welcome to the catalog for my course visualization project. In this project I created an interactive dependency graph for all the classes in each of the majors at MIT. Hovering over any class on a graph shows you the class title, while clicking on it brings you to the class's description.

Select a major to check it out.

Development Details

I was motivated to complete this project by the frustrating amount of backtracking required to see the prereqs for advanced classes at MIT. MIT's course catalog only lists the direct prereqs for each class, yet those prereqs often have prereqs as well. I wanted a way to visualize the "depths" of classes, so I created this tool.

This project was created entirely in Python. I used the webscraping library Beautiful Soup to scrape MIT's Course Catalog, the graphing library iGraph to create the abstract graphs for each major, and the plotting library Plotly to visualize the graphs and store them as HTML files. You can find my Github repository for this project here.

Further Graph Details

Some simplifications were made to this model. Some classes in a major depend on classes outside of that major, and those dependencies were ignored for visual simplicity. For example, the dependency of 8.04 Quantum Physics I (a physics course) on 18.03 Differential Equations (a math course) was not shown. Also, some more advanced classes only require "permission of instructor." Those classes were also ignored because they have no implementable dependency.

The graphs vary in aesthetics. For some majors, almost all of the classes have no prereqs or only require instructor permission. I completely excluded Course 21 for this reason, as it's graph was essentially empty. I also didn't include Course 3: Materials Science, since many of its listed classes have prereqs that don't exist. Some majors have so many classes that the graphs can be hard to read without zooming (e.g. Course 6, 18). Other majors don't have a central introductory course (e.g. Courses 11, 12) so their graphs are less unified.
